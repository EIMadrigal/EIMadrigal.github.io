微软实习笔试 2.26

第一题： 给一个string包括^v<>四种字符，分别表示四个方向的移动，不能忘诱人的地方东，问最后有多少个字符可以顺利移动。 N = 50

第二题： N = 10^5 给定一个tag数组S: A-Z 还有点的横纵坐标X Y，每个点都有一个tag，求以原点为圆心的圆最多可以扩多大，要求圆里不能含有相同tag的点，在圆周上也算圆内。问圆内最多能有多少个点。

第三题： N = 10^5 给一个vector，要求把该数组分为3段，选择两个不相邻的分裂点i j使得 nums[i] + nums[j]最小。
