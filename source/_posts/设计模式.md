## 单例模式
顾名思义就是某些类全局只需要一个实例。

 1. 饿汉模式
  只要系统开始运行便创建实例，不管是否需要，需要时直接调用。
  
 2. 懒汉模式

```cpp
class Singleton {
public:
    static const Singleton* getInstance() {
        return instance;
    }
private:
    // 禁止外部构造、析构、拷贝、赋值
    Singleton() {}
    ~Singleton() {}
    Singleton(const Singleton& s);
    Singleton& operator=(const Singleton& s);
    static const Singleton* instance;
};
```
