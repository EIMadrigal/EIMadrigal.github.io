笔试注意点

Posted on 2021-10-16 In Computer Science Views:
踩坑血泪...

输入输出

  1. cin的结束符是空格, 制表符和回车, 读完后结束符仍然在缓冲区. getline的结束符是回车, 读完后结束符不在缓冲区. 读取一行有空格字符串需要getline.
  1. 将空格分隔的字符串读到vector
  1. 知道数组大小后，可以直接读到数组中
  1. 删除首尾空格

STL总结

  1. int stoi(const string& str, size_t* idx=0, int base=10)：idx指向第一个非数字字符
  2. isalpha(int ch): 头文件cctype
  3. 对map或unordered_map按值排序
  4. unordered_map自定义key
    unordered_map<pair<int, int>, int>会报错，需要提供一个hash函数. 如果key不存在，访问仍然成功，返回value对象默认构造值 但是map就不需要, map的key类型必须支持<运算符
  5. sort(v.begin(), v.end(), [](int a, int b) { return a > b; })或sort(v.rbegin(), v.rend())
  6. vector追加: dest.insert(dest.end(), src.begin(), src.end()). vector<vector<int>>每一维元素个数可以不同
  7. C++ 17的structured binding: for (auto [k, v] : unordered_map)
  8. std::lower_bound和std::upper_bound对于random-access iterators的时间复杂度是\(O(lgn)\)，但对于non-random-access iterators如set::iterator是\(O(n)\)的。这种情况下应使用set自带的setObj.upper_bound(target)，复杂度\(O(lgn)\)
